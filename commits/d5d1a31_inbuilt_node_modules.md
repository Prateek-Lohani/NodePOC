## Title: Inbuilt node.js module (util module)

## Description
This commit explores Node.js built-in utility modules, particularly the `util` module.

## Concept Learned
- Node.js built-in modules
- Using util module for debugging
- Module introspection

## Code Example
```
javascript
const util = require('node:util');
const fs = require('node:fs');

console.log(util.inspect(fs.readdirSync('.')));
```

## Key Takeaway
Node.js provides many built-in modules that don't require external installation.
